[tenki.jp](https://tenki.jp/ "日本気象協会 tenki.jp / 天気予報・地震・台風") で「紅葉見頃情報」を見ると大根島にある{{% ruby "ゆうしえん" %}}[由志園]{{% /ruby %}}の紅葉が見頃とあったので，運動がてら行ってみることにした。


## 大根島遊覧

まずはカフェインをドーピング。

{{< fig-img src="./54925413642_be0dc1d071_e.jpg" title="コメダでカフェインをドーピング中 | Flickr" link="https://www.flickr.com/photos/spiegel/54925413642/" >}}

気合が入ったところで移動を開始する。
今日はピーカンに晴れていて，ほぼ無風。

{{< fig-img src="./54926822568_55117d37a5_e.jpg" title="天高く... | Flickr" link="https://www.flickr.com/photos/spiegel/54926822568/" >}}

最近お気に入りのコース，大橋川の北側を東進する。

{{< fig-img src="./54926823683_826a89d21d_e.jpg" title="大橋川 | Flickr" link="https://www.flickr.com/photos/spiegel/54926823683/" >}}

微妙に視程が悪い。
大山が霞んで見える。

{{< fig-img src="./54925754162_46f7597a70_e.jpg" title="大山と中海大橋 | Flickr" link="https://www.flickr.com/photos/spiegel/54925754162/" >}}

ほどなく大根島に入ったのだが，ほとんどノープランで出てきてしまったので，まずは南岸を反時計回りに走ってみる。
おっ[幽鬼洞]なんてのがあるのか。

{{< fig-img src="./54926827593_37d948d1dc_e.jpg" title="熔岩洞窟 幽鬼洞へ | Flickr" link="https://www.flickr.com/photos/spiegel/54926827593/" >}}

国の特別天然記念物らしい。
行ってみるか。

{{< fig-img src="./54926931365_d7c60cae34_e.jpg" title="熔岩洞窟 幽鬼洞（立ち入り禁止） | Flickr" link="https://www.flickr.com/photos/spiegel/54926931365/" >}}

おおぅ。
一般人は立入禁止らしい。
残念。
入り口だけ撮っておくか。

{{< fig-img src="./54926829778_a520f5a93d_e.jpg" title="熔岩洞窟 幽鬼洞 | Flickr" link="https://www.flickr.com/photos/spiegel/54926829778/" >}}

大根島の地下には，こうした熔岩トンネルが縦横無尽に走っているらしい。

## 由志園へ

[幽鬼洞]を後にして，大根島を適当に散策する。

{{< fig-img src="./54926934380_3e48278133_e.jpg" title="大根島 | Flickr" link="https://www.flickr.com/photos/spiegel/54926934380/" >}}

そのうち[由志園]の案内板が見えてきたので，そちらへ向かう。
到着した。

{{< fig-img src="./54925661317_1557e102f4_e.jpg" title="由志園 入口 | Flickr" link="https://www.flickr.com/photos/spiegel/54925661317/" >}}

自転車用の駐輪場もあった。
ありがたや。
まぁ，私の他に自転車を駐めてる人はいなかったけどね（笑）

入場料は大人1人1,200円（税込）。

{{< fig-img src="./54926910576_3a4733c3ff_e.jpg" title="由志園 入場チケット | Flickr" link="https://www.flickr.com/photos/spiegel/54926910576/" >}}

このチケットがあれば，同日内は再入場できるらしい。

それでは紅葉狩りといきましょう。

{{< fig-img src="./54926534606_09d7952b82_e.jpg" title="紅葉狩り（由志園） | Flickr" link="https://www.flickr.com/photos/spiegel/54926534606/" >}}

{{< fig-img src="./54926537376_e3665f2081_e.jpg" title="紅葉狩り（由志園） | Flickr" link="https://www.flickr.com/photos/spiegel/54926537376/" >}}

{{< fig-img src="./54926844090_f561151fc3_e.jpg" title="紅葉狩り（由志園） | Flickr" link="https://www.flickr.com/photos/spiegel/54926844090/" >}}

{{< fig-img src="./54926744668_033c7caa9e_e.jpg" title="由志園 庭園 | Flickr" link="https://www.flickr.com/photos/spiegel/54926744668/" >}}

やっぱ，きちんと手入れされている庭園の紅葉は綺麗だわ。

庭園をぐるっと散策して入り口の建物に戻る。
喫茶店があるので一休み。

{{< fig-img src="./54926751323_37bcafcb4f_e.jpg" title="由志園 茶房「一望」 | Flickr" link="https://www.flickr.com/photos/spiegel/54926751323/" >}}

お庭を眺めながら濃茶とぜんざいをいただく。

{{< fig-img src="./54926752103_6aa8265431_e.jpg" title="濃茶とぜんざい（由志園 茶房「一望」） | Flickr" link="https://www.flickr.com/photos/spiegel/54926752103/" >}}

いやぁ。
楽しかった。

## 道の駅本庄を経由して帰宅

まっすぐ市内に戻るのもアレなので，[道の駅本庄]に寄ってみた。
うわ。
吊し柿のまま売ってるのか？

{{< fig-img src="./54926937290_7416cd2b9b_e.jpg" title="吊し柿のまま売ってる（道の駅本庄） | Flickr" link="https://www.flickr.com/photos/spiegel/54926937290/" >}}

これはスルーして，いつものやつをいただく。

{{< fig-img src="./54926885049_b4b025b6fb_e.jpg" title="柿ソフトクリーム（道の駅本庄、50円値上げ） | Flickr" link="https://www.flickr.com/photos/spiegel/54926885049/" >}}

柿の味がちょっと濃くなった？ でも50円の値上がり。
うんうん。
政治が悪い（笑）

ほんじゃあ，市内に戻りますかね。
いったん[図書館][島根県立図書館]まで戻って夕方まで読書。
それから帰宅。

{{< fig-img src="./54927086229_880303a167_e.jpg" title="宍道湖 夕景（嫁ヶ島） | Flickr" link="https://www.flickr.com/photos/spiegel/54927086229/" >}}

お疲れ様でした。
